My Services

Aircraft/Spacecraft Simulation Development

Aircraft and spacecraft simulation design in Simulink/MATLAB/C++. Modelling requires stability and control derivatives (OpenVSP, AVL) along with A/C mass, CG and CM location. For unmanned aircraft, controllers are designed using classic control theory or LQR.

Hardware Driver Development for Aerospace Applications

Driver development in C++ 14/17 for different hardware, including ARINC429-based inceptors (BAE SYSTEMS) or UDP-based control frames (BRUNNER).

Learn more

3D Visualization Plugins for X-Plane and FlightGear

Often simulations require 3D visualization. Ad-Hoc Plugins are developed using two main software programs for 3D rendering leverage: X-Plane and/or FlightGear.

Model-based Design (MBD) of an F-16 aircraft

This mix of C++ 14 code and Simulink models allows to compile an F-16 simulator into a real-time time simulation executable. The origin model files have been adapted from a Simulink simulation file available from Dr. Balas' University of Minnesota personal page.
